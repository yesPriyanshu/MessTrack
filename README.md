# MessTrack — IIEST Shibpur AI Hostel Mess Management Portal

A frontend-only mess management prototype built for IIEST Shibpur, covering all hostels, halls, and campus canteens. Features include daily menu tracking, food waste indicators, mess ratings, monthly expense dashboards, and an AI-powered review system via n8n that sends student feedback to Google Sheets and generates daily meal quality reports using GPT-4o. The JMCR admin panel allows authorized committee members to publish live menus and split expenses across students.
